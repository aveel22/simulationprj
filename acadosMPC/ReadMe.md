# Installation
Acados: https://docs.acados.org/installation/

1. Clone acados

```
git clone https://github.com/acados/acados.git
cd acados
git submodule update --recursive --init
```
2. run in terminal under correct python version
```
python -m pip install -e C:\acados\interfaces\acados_template
```
3. It should work
